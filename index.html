TacoSpigot

TacoSpigot is a even higher performance fork of PaperSpigot.

Features

  • Optimizes hoppers by 80-95
    • Adds an option to disable InventoryMoveItemEvent
  • Optimizes armor stands by 90%+
    • Disables gravity for armor stands (you must set "armor-stand.optimize-movement" to true)
    • Avoiding looking up entities when moving
  • XRay optimization
    • Reduces redundant checking for blocks
  • Optimize getting blocks
    • Helps with entities and redstone
    • Doesn't even show up in the developer's samplings anymore
  • Optimize setting blocks
    • Helps with redstone
    • Is now at '0%' in samplings
  • Slightly faster event execution

Downloading

TacoSpigot has pre-built binaries built using Paperclip You can download them here: https://techcable.net/downloads/tacospigot/

There are 2 types of releases)

  • The versions are marked 'stable' if they are considered stable.
    • Most server owners should choose this
    • These builds usually have been in production for a while
  • The versions are marked 'semi-stable', if they are considered 'almost' stable
    • Choose this if you wan't the latest performance optimization, and are fine with having some potential bugs
    • This probably hasn't been in production very long
    • These builds will always have gone through testing on the developer's machine
    • You should probably still test these on your own though, as I may miss something, or not do very thorough testing.

Unless I get a CI server, I'm not goona put up development builds.

Building

  • Download TacoTools and run it.
    • Make sure to select TacoTools.jar
  • Find the tacospigot jar in "output"
    • Named taco-{minecraft version}.jar
