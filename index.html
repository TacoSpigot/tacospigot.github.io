TacoSpigot

TacoSpigot is a even higher performance fork of PaperSpigot.

Features

  • Optimizes hoppers by 80-95
    • Adds an option to disable InventoryMoveItemEvent
  • Optimizes armor stands by 90%+
    • Disables gravity for armor stands (you must set "armor-stand.optimize-movement" to true)
    • Avoiding looking up entities when moving
  • Improve removing entities
    • This can actually provide a noticeable speedup for lots of entities
    • Went from O(n^2) time to O(n) time
    • Credits to Aikar

Building

  • Download TacoTools and run it.
    • Make sure to select TacoTools.jar
  • Find the tacospigot jar in "output"
    • Named taco-{minecraft version}.jar
